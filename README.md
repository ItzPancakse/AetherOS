# AetherOS
my computer craft os
linux like os for CC:Tweaked
meant to control stuff in Create mod like trains and machines.

logo from https://opengameart.org/content/cloud


## How to install!
On your computer in CC: Tweaked run
```shell
wget run https://raw.githubusercontent.com/ItzPancakse/AetherOS/main/netinstall.lua
```
Hope that helps!

### What if my server admin hasn't enabled HTTP?
You can ask them to enable it or copy all the files in this repo to the computer in your server
